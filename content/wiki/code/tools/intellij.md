#### De meest voorkomende shortcuts vergelijken 

| **Betekenis** | **Eclipse shortcut** | **IntelliJ shortcut** |
| Open Type | CTRL+SHIFT+T | CTRL+N |
| Close Editor | CTRL+W | CTRL+F4 |
| Delete line | CTRL+D | CTRL+Y |
